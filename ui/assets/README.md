# Brand assets

Drop your official **Canada Basketball logo** here and the app uses it
everywhere (header, sidebar, empty state, and browser favicon).

- Filename: `logo.svg` (preferred), or `logo.png` / `logo.jpg` / `logo.webp`
- A wide or square lockup both work; it's scaled to height and width auto.
- No restart needed — refresh the app after adding the file.

The official Canada Basketball logo is **trademarked** and is intentionally not
bundled with this project. Until you add your licensed file, the app shows an
original maple-leaf-on-basketball mark (`ui/theme.py: maple_mark`).
